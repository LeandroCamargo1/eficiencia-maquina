# 🚀 INÍCIO RÁPIDO - AppBJJ Kids

## ⚡ 30 SEGUNDOS PARA COMEÇAR

### 1. Abra o App
```
Opção A: Duplo clique em index.html
Opção B: Clique direito → Abrir com navegador
Opção C: Arraste index.html para o navegador
```

### 2. Crie um Perfil
```
Clique em 👤 (canto superior direito)
Clique em "+ Novo Perfil"
Digite: Nome e Idade
Clique em "Criar Perfil"
```

### 3. Marque sua Primeira Aula
```
Clique em um dia no calendário
Veja o dia ficar vermelho
Veja o progresso aumentar
```

### 4. Explore
```
Clique em ☀️ para trocar tema
Veja as estatísticas
Clique em ⚙️ para configurar
Confira os badges ganhos
```

**Pronto! Você está usando AppBJJ Kids! 🎉**

## 🐛 Algo não Funciona?

### Problema: Dados não salvam
```
Solução:
1. Abra DevTools (F12)
2. Application → Local Storage
3. Procure por "appbjj-kids-state-v2"
4. Deve ter dados salvos
5. Se não, limpe cookies e tente novamente
```

### Problema: Gráfico não aparece
```
Solução:
1. Verifique internet (precisa de CDN)
2. Abra DevTools → Console
3. Procure por erros de Chart.js
4. Se tiver erro de rede, recarregue (F5)
```

### Problema: PWA não instala
```
Solução:
1. Acesse online via HTTPS (Vercel/Netlify)
2. Aguarde carregar (instala Service Worker)
3. Clique em instalar (browser vai oferecer)
4. Pronto, agora pode usar offline
```

### Problema: Interface fica travada
```
Solução:
1. Recarregue página (F5)
2. Limpe cache (Ctrl+Shift+Delete)
3. Tente em modo privado
4. Se persistir, relate o erro
```

### Problema: Não consegue criar perfil
```
Solução:
1. Verifique console (F12 → Console)
2. Veja se há mensagem de erro
3. Certifique-se de preencher nome e idade
4. Clique novamente
```
